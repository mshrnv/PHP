Задание 10

Гостевая книга (версия 3)
+ в виде класса
+ с шаблонами страниц

Разместить код в файле index.php
CSS разместить в design/style.css
картинки разместить в design/images/
файл сообщений: data/guestbook.txt
файл с пользователями: data/users.txt
классы разместить в lib/ClassName.php
шаблоны разместить в template/TemplateName.tpl

Необходимо переписать гостевую книгу 2 с помощью класса GuestBook.php.
Идею реализации класса взять тут:
https://svnhub.ru/soft/PHP%20books/%5bLaura_Tomson,_Lyuk_Velling%5d_Razrabotka_Web-priloz(BookSee.org).pdf
Листинг 6.1 (стр. 151) и в файле lections/my_blog.zip
 
Необходимые условия сдачи (кроме требований к Гостевой книге 2):
- разбить всю страницу на блоки, для каждого блока реализовать шаблон на языке класса Template.php (размещать в папке templates) и функцию вывода в классе GuestBook
- в классе GuestBook реализовать метод show() который собирает полный HTML всей гостевой книги из блоков.
- в классе GuestBook реализовать метод route(), который обрабатывает входящий request для обработки различных состояний гостевой книги.
- правильно использовать сокрытие методов и свойств.
- параметры гостевой книги хранить в свойствах класса GuestBook с возможностью установки их через конструктор или через метод-setter.
